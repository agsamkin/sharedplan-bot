## ADDED Requirements

### Requirement: README.md с описанием проекта

В корне репозитория ДОЛЖЕН существовать файл `README.md` на русском языке с описанием проекта Shared Plan Bot, его назначения и ключевых возможностей.

#### Scenario: README содержит описание проекта
- **WHEN** пользователь открывает README.md
- **THEN** файл содержит заголовок проекта, краткое описание (Telegram-бот для совместного планирования событий) и список ключевых возможностей (spaces, события, напоминания, Mini App, AI-парсинг, голосовой ввод)

### Requirement: Пометка демо-проекта и AI

README ДОЛЖЕН содержать заметную секцию, указывающую что это демо-проект, реализованный с использованием AI — Claude Opus 4.6 и OpenSpec.

#### Scenario: README содержит пометку о демо и AI
- **WHEN** пользователь читает README.md
- **THEN** файл содержит явное указание что проект является демонстрационным и создан с помощью AI (Claude Opus 4.6, OpenSpec)

### Requirement: Инструкции по запуску

README ДОЛЖЕН содержать секцию Quick Start с минимальными шагами для запуска проекта через Docker Compose.

#### Scenario: Quick Start содержит шаги запуска
- **WHEN** пользователь следует инструкциям Quick Start
- **THEN** секция описывает: клонирование репозитория, копирование `.env.example` в `.env`, заполнение обязательных переменных, запуск `docker compose up`

### Requirement: Таблица переменных окружения

README ДОЛЖЕН содержать таблицу со всеми переменными окружения: имя, обязательность, значение по умолчанию, описание.

#### Scenario: Таблица содержит все переменные
- **WHEN** пользователь ищет информацию о переменных окружения
- **THEN** таблица содержит обязательные переменные (`TELEGRAM_BOT_TOKEN`, `OWNER_TELEGRAM_ID`), переменные БД (`DATABASE_URL` или `DB_USER`/`DB_PASSWORD`/`DB_HOST`/`DB_PORT`/`DB_NAME`), опциональные API-ключи и настройки с дефолтами

### Requirement: Секция лицензии

README ДОЛЖЕН содержать секцию с указанием лицензии MIT и ссылкой на файл LICENSE.

#### Scenario: README ссылается на лицензию
- **WHEN** пользователь ищет информацию о лицензии
- **THEN** README содержит секцию «Лицензия» со ссылкой на файл LICENSE и указанием MIT
