## ADDED Requirements

### Requirement: Регистрация команд при старте бота

Система ДОЛЖНА регистрировать список пользовательских команд через Telegram Bot API (`set_my_commands`) при каждом запуске бота. Команды ДОЛЖНЫ отображаться в меню Telegram (кнопка «/» рядом с полем ввода).

#### Scenario: Успешная регистрация команд при старте

- **WHEN** бот запускается
- **THEN** вызывается `bot.set_my_commands()` с полным списком пользовательских команд
- **THEN** команды отображаются в меню Telegram для всех пользователей

### Requirement: Состав списка команд

Список зарегистрированных команд ДОЛЖЕН включать все публичные пользовательские команды бота: `start`, `help`, `newspace`, `spaces`, `space_info`, `events`, `reminders`. Каждая команда ДОЛЖНА иметь краткое описание на русском языке.

#### Scenario: Полный список команд в меню

- **WHEN** пользователь открывает меню команд бота в Telegram
- **THEN** отображаются все 7 команд с описаниями на русском языке

### Requirement: Централизованное определение списка команд

Список команд ДОЛЖЕН быть определён в одном месте (`app/bot/commands.py`) как переиспользуемая константа. Модуль `main.py` ДОЛЖЕН импортировать список из этого модуля.

#### Scenario: Единый источник правды для команд

- **WHEN** разработчик добавляет новую команду
- **THEN** достаточно добавить её в один файл `app/bot/commands.py` для обновления меню
