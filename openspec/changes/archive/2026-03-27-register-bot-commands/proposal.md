## Why

Сейчас бот не регистрирует свои команды через Telegram Bot API (`set_my_commands`). Из-за этого пользователи не видят список доступных команд в меню Telegram (кнопка «/» рядом с полем ввода), что снижает discoverability и удобство использования бота.

## What Changes

- При старте бота вызывается `bot.set_my_commands()` с актуальным списком пользовательских команд
- Список команд определяется централизованно в одном месте для консистентности с хендлерами

## Capabilities

### New Capabilities

- `bot-commands-menu`: Регистрация команд бота в Telegram при старте для отображения в меню

### Modified Capabilities

Нет изменений в существующих спецификациях.

## Impact

- `app/main.py` — добавление вызова `set_my_commands` при инициализации бота
- Зависимость от Telegram Bot API (метод `setMyCommands`) — уже доступен через aiogram
