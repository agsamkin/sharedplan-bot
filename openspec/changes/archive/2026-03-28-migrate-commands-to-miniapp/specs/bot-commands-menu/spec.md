## MODIFIED Requirements

### Requirement: Состав списка команд

Список зарегистрированных команд ДОЛЖЕН содержать только минимальный набор: `help` — «Справка», `privacy` — «Политика конфиденциальности». Команды `start`, `app`, `newspace`, `spaces`, `space_info`, `events`, `reminders` НЕ ДОЛЖНЫ регистрироваться. Вместо команды `/app` ДОЛЖНА использоваться MenuButtonWebApp (кнопка меню, открывающая Mini App).

#### Scenario: Минимальный список команд в меню
- **WHEN** пользователь открывает меню команд бота в Telegram
- **THEN** отображаются команды: `help` — «Справка», `privacy` — «Политика конфиденциальности»

#### Scenario: Кнопка меню открывает Mini App
- **WHEN** пользователь нажимает кнопку меню (слева от поля ввода)
- **THEN** открывается Mini App через MenuButtonWebApp

### Requirement: Улучшенное приветственное сообщение /start

Обработчик `/start` ДОЛЖЕН отправлять приветственное сообщение с кратким описанием возможностей бота и inline-кнопкой WebApp для открытия Mini App. Сообщение ДОЛЖНО направлять пользователя в Mini App для управления пространствами и событиями. При создании события ДОЛЖНА быть инструкция отправить текст или голосовое сообщение.

При наличии deep link payload с префиксом `join_` обработчик ДОЛЖЕН сначала обработать присоединение, затем показать приветствие.

#### Scenario: Новый пользователь запускает бота
- **WHEN** пользователь впервые отправляет `/start`
- **THEN** бот отвечает приветственным сообщением, включающим: краткое описание возможностей (планирование событий, текст/голос, напоминания), инструкцию открыть Mini App для создания пространства, inline-кнопку WebApp «Открыть приложение»

#### Scenario: Пользователь с существующими пространствами запускает бота
- **WHEN** пользователь, уже состоящий в пространствах, отправляет `/start` (без deep link)
- **THEN** бот отвечает приветственным сообщением с inline-кнопкой WebApp «Открыть приложение»

### Requirement: Улучшенная справка /help

Обработчик `/help` ДОЛЖЕН показывать краткую справку с описанием работы бота. Справка ДОЛЖНА объяснять: как создавать события (отправить текст или голосовое сообщение), как управлять пространствами и событиями через Mini App (кнопка меню), список оставшихся команд (/help, /privacy).

#### Scenario: Пользователь запрашивает помощь
- **WHEN** пользователь отправляет `/help`
- **THEN** бот отвечает справкой: инструкция по созданию событий (текст/голос), упоминание кнопки меню для Mini App, список команд (help, privacy)

## REMOVED Requirements

### Requirement: Команда /app для открытия Mini App
**Reason**: Команда /app заменена на MenuButtonWebApp — кнопку меню слева от поля ввода, которая открывает Mini App напрямую. Хэндлер /app удалён.
**Migration**: Пользователи открывают Mini App через кнопку меню в интерфейсе Telegram.
