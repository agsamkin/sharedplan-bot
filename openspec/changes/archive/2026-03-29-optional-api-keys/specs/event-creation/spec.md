## ADDED Requirements

### Requirement: Обработка отключённого LLM при текстовом вводе

При получении текстового сообщения, если LLM-парсинг отключён (`service_disabled`), бот ДОЛЖЕН отправить пользователю понятное сообщение с рекомендацией использовать Mini App.

#### Scenario: OpenRouter отключён — текстовое сообщение

- **WHEN** пользователь отправляет текстовое сообщение и `parse_event()` выбрасывает `ParseError` с типом `service_disabled`
- **THEN** бот отвечает: «Автоматическое распознавание событий отключено. Создай событие вручную в приложении.»

#### Scenario: OpenRouter отключён — кнопка Mini App

- **WHEN** пользователь отправляет текстовое сообщение, `parse_event()` выбрасывает `ParseError` с типом `service_disabled`, и `MINI_APP_URL` задан в конфигурации
- **THEN** к сообщению добавляется inline-кнопка «Открыть приложение» со ссылкой на Mini App
