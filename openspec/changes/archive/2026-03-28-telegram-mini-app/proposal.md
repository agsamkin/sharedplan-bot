## Why

Управление событиями и пространствами через inline-кнопки Telegram бота ограничено: нет удобных списков, невозможно быстро редактировать поля, нет визуального календаря. Telegram Mini App позволяет создать полноценный веб-интерфейс внутри Telegram, сохраняя бесшовный UX.

## What Changes

- **Новый веб-сервер** для Mini App (бэкенд + статика), работающий рядом с ботом в том же Docker Compose стеке
- **REST API** для операций с событиями и пространствами, использующий существующие сервисы (`event_service`, `space_service`, `reminder_service`)
- **Telegram Mini App фронтенд** — SPA для управления событиями, пространствами и настройками напоминаний
- **Авторизация через Telegram WebApp** — валидация `initData` от Telegram на бэкенде
- **Бот остаётся для минимума**: создание событий (текст/голос), создание пространств, присоединение по ссылке, получение напоминаний
- **Удаление из бота** управления списками событий, редактирования/удаления событий, управления участниками пространств, удаления пространств — всё это переносится в Mini App
- **Кнопка открытия Mini App** в боте вместо текущих команд управления

## Capabilities

### New Capabilities
- `mini-app-backend`: HTTP-сервер (FastAPI/aiohttp), REST API эндпоинты для событий, пространств и настроек, Telegram WebApp авторизация
- `mini-app-frontend`: SPA-интерфейс Mini App — списки событий, редактирование, управление пространствами и участниками, настройки напоминаний

### Modified Capabilities
- `event-management`: редактирование и удаление событий переносится из Telegram callback-ов в Mini App API; бот-хендлеры упрощаются
- `events-list`: просмотр списка событий переносится в Mini App; в боте остаётся только кнопка «Открыть Mini App»
- `space-management`: управление участниками (kick), удаление пространств переносится в Mini App; в боте остаётся только `/newspace` и `/spaces` (со ссылкой на Mini App)
- `reminder-settings`: настройка интервалов напоминаний переносится из inline-кнопок бота в Mini App UI
- `bot-commands-menu`: обновление списка команд — убрать управляющие команды, добавить открытие Mini App
- `telegram-bot-core`: добавление веб-сервера в lifecycle приложения, конфигурация Mini App URL
- `docker-environment`: добавление сборки фронтенда и раздачи статики, новые переменные окружения

## Impact

- **Новые зависимости**: FastAPI (или aiohttp web), фронтенд-фреймворк (React/Vue/Svelte), Telegram WebApp JS SDK
- **Код бота**: удаление/упрощение хендлеров управления событиями, списков, настроек напоминаний, управления пространствами
- **Инфраструктура**: Docker Compose получает сборку фронтенда, возможно nginx для статики, необходим HTTPS для Mini App
- **БД**: схема не меняется — Mini App использует те же сервисы и модели
- **Конфигурация**: новые env-переменные для Mini App URL, HTTPS-сертификатов
