## MODIFIED Requirements

### Requirement: Состав списка команд

Список зарегистрированных команд ДОЛЖЕН быть обновлён: убраны команды управления (их функции доступны в Mini App), добавлена команда открытия Mini App.

#### Scenario: Обновлённый список команд в меню
- **WHEN** пользователь открывает меню команд бота в Telegram
- **THEN** отображаются команды: `start` — «Начать работу», `help` — «Список команд и помощь», `newspace` — «Создать пространство», `spaces` — «Мои пространства», `events` — «Ближайшие события», `app` — «Открыть приложение»

### Requirement: Команда /app для открытия Mini App

Обработчик `/app` ДОЛЖЕН отправлять inline-кнопку с WebApp URL для открытия Mini App.

#### Scenario: Пользователь отправляет /app
- **WHEN** пользователь отправляет `/app`
- **THEN** бот отвечает сообщением с inline-кнопкой WebApp «📱 Открыть приложение»

## REMOVED Requirements

### Requirement: Команда /reminders в меню
**Reason**: Настройки напоминаний доступны через Mini App. Команда `/reminders` перенаправляет в Mini App.
**Migration**: Используется `/app` или прямая ссылка из `/reminders`.

### Requirement: Команда /space_info в меню
**Reason**: Информация о пространстве доступна в Mini App.
**Migration**: Используется страница пространства в Mini App.
