## MODIFIED Requirements

### Requirement: Confirmation card
Карточка подтверждения ДОЛЖНА отображать: `📌 Событие:\n📝 {title}\n📅 {date}, {weekday}\n⏰ {time}` (время только если есть). Если событие повторяющееся — добавить строку `🔄 {repeat_label}` (например "Каждую неделю", "Каждый месяц"). Три кнопки: `[✅ Да] [✏️ Изменить] [❌ Отмена]`. Для голосовых — вверху `🎤 Распознано: "{transcript}"`. Предупреждение о конфликте времени (±2ч на ту же дату) если применимо. Предупреждение о прошлой дате имеет приоритет над конфликтами.

#### Scenario: Карточка повторяющегося события
- **WHEN** LLM-парсер вернул событие с `recurrence_rule = "weekly"`
- **THEN** карточка подтверждения содержит строку `🔄 Каждую неделю`

#### Scenario: Карточка одноразового события
- **WHEN** LLM-парсер вернул событие с `recurrence_rule = null`
- **THEN** карточка подтверждения НЕ содержит строку с повторением

### Requirement: Event confirmation
При подтверждении система ДОЛЖНА: создать запись `events` (с `recurrence_rule` если указан), создать `scheduled_reminders` для всех участников пространства (с учётом `reminder_settings`, пропуск если `remind_at` в прошлом), отправить уведомления участникам (кроме создателя). Если событие повторяющееся — сгенерировать дочерние вхождения на 60 дней вперёд и создать напоминания для каждого вхождения.

#### Scenario: Подтверждение повторяющегося события
- **WHEN** пользователь нажимает "✅ Да" для повторяющегося события
- **THEN** создаётся родительское событие, генерируются дочерние вхождения на 60 дней, для каждого создаются напоминания

#### Scenario: Уведомления о повторяющемся событии
- **WHEN** публикуется повторяющееся событие
- **THEN** уведомление участникам содержит информацию о повторении (например "🔄 Каждую неделю")

### Requirement: Notifications
Уведомление участникам (кроме создателя) ДОЛЖНО содержать: заголовок, дату (расширенный относительный формат: сегодня, завтра, послезавтра, день недели до 6 дней, числовой формат далее), время (если есть), информацию о повторении (если есть). Формат повторения: "🔄 Каждый день", "🔄 Каждую неделю", "🔄 Каждые 2 недели", "🔄 Каждый месяц", "🔄 Каждый год".

#### Scenario: Уведомление с повторением
- **WHEN** создано повторяющееся событие с `recurrence_rule = "monthly"`
- **THEN** уведомление содержит строку "🔄 Каждый месяц"
