# Capability: reminder-settings

## Purpose

Управление персональными настройками напоминаний пользователя: команда `/reminders` направляет в Mini App, где доступны полноценные toggle-переключатели для интервалов напоминаний (1d, 2h, 1h, 30m, 15m, 0m).

## Requirements

### Requirement: Команда /reminders — ссылка на Mini App

Обработчик команды `/reminders` ДОЛЖЕН направлять пользователя в Mini App для настройки напоминаний.

#### Scenario: Пользователь вызывает /reminders
- **WHEN** пользователь отправляет команду `/reminders`
- **THEN** бот отвечает сообщением «Настрой напоминания в приложении:» с inline-кнопкой WebApp «⏰ Настройки напоминаний» со ссылкой на Mini App (маршрут `/settings/reminders`)

### Requirement: Чтение настроек с forward-compatibility

При чтении `reminder_settings` из БД код ДОЛЖЕН использовать `.get(key, False)` для каждого ключа, чтобы отсутствующие ключи трактовались как `False`.

#### Scenario: Пользователь с неполным JSONB
- **WHEN** у пользователя в `reminder_settings` отсутствует ключ `0m`
- **THEN** `.get("0m", False)` возвращает `False`, интерфейс отображает `☐ В момент события`

### Requirement: Метки интервалов на русском языке

Каждый интервал напоминания ДОЛЖЕН иметь локализованную метку на языке пользователя:

Русский (`ru`):
- `1d` → «За 1 день»
- `2h` → «За 2 часа»
- `1h` → «За 1 час»
- `30m` → «За 30 минут»
- `15m` → «За 15 минут»
- `0m` → «В момент события»

Английский (`en`):
- `1d` → «1 day before»
- `2h` → «2 hours before»
- `1h` → «1 hour before»
- `30m` → «30 min before»
- `15m` → «15 min before»
- `0m` → «At the time of event»

Метки ДОЛЖНЫ использоваться как в боте (уведомления, команды), так и в Mini App (экран настроек напоминаний).

#### Scenario: Маппинг ключей к меткам
- **WHEN** отображается интерфейс настроек
- **THEN** используются метки: `1d` → «За 1 день», `2h` → «За 2 часа», `1h` → «За 1 час», `30m` → «За 30 минут», `15m` → «За 15 минут», `0m` → «В момент события»

#### Scenario: Метки на русском
- **WHEN** пользователь с `language = "ru"` открывает экран настроек напоминаний
- **THEN** интервалы отображаются с русскими метками: "За 1 день", "За 2 часа" и т.д.

#### Scenario: Метки на английском
- **WHEN** пользователь с `language = "en"` открывает экран настроек напоминаний
- **THEN** интервалы отображаются с английскими метками: "1 day before", "2 hours before" и т.д.
